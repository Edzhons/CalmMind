# CalmMind (MVP)

## FOLLOWING INFO TO BE UPDATED.....

**CalmMind** is a simple desktop app for personal mental health tracking. This MVP version allows users to track daily entries, set reminders, and view past records. Built with Python and Tkinter.

---

## Features
- Add, edit, and delete daily entries
- Set reminders for tasks or reflections
- View past entries in a calendar view
- Lightweight and easy to use
